1. Connect to tensorflow inception v3 pod

$ kubectl get pods
$ kubectl exec -it inception -- bash

2. Run query versus the server

Download an image and run the query

$ curl -o 'cat-over-keyboard.jpg' 'http://www.rd.com/wp-content/uploads/sites/2/2016/04/01-cat-wants-to-tell-you-laptop.jpg'
$ bazel-bin/tensorflow_serving/example/inception_client --server=localhost:9000 --image=cat-over-keyboard.jpg
